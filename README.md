## 🚀 Overview

**AutoCraft** is built to streamline software engineering tasks by leveraging AI agents, automated workflows, and intelligent tools. Whether it's scaffolding projects, writing code, or running diagnostics, AutoCraft crafts solutions autonomously.

---

## 🛠️ Features

* **Autonomous Workflows:** Execute multi-step software development tasks.
* **Modular Architecture:** Easily plug in custom tools, LLM providers, and parsers.
* **Developer-Centric:** Designed with clean configurations and robust testing in mind.
